# SatriaNFT
## Instalasi
### Instal dependencies
`
npm install
`
### Jalankan di mesin lokal
Compile frontend:
`
npm run dev
`
Buka website:
2 Cara:
`Gunakan extension live server` atau `Paste index.html path ke browser`
### Buka aplikasi di browser
`
Masuk ke localhost:300
`
